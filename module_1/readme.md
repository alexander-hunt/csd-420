# Random Cards JavaFX Application

This JavaFX program displays four randomly selected playing cards from a deck of 52. A **Refresh** button allows you to generate a new set of cards.

---

## Requirements

- Java JDK
- JavaFX SDK

---

## Compile and Run (PowerShell)

Open PowerShell in the module_1 directory and execute:

```powershell
javac --module-path "C:\path\to\javafx-sdk-26\lib" --add-modules javafx.controls RandomCardsFX.java
java --module-path "C:\path\to\javafx-sdk-26\lib" --add-modules javafx.controls RandomCardsFX
