# Module 3: Remove Duplicates from ArrayList

This module contains a test program that demonstrates removing duplicates from an ArrayList while preserving the order of first occurrences.

## Files
- `TestRemoveDuplicates.java`: Contains the `removeDuplicates` method and a main method that fills an ArrayList with 50 random integers (1-20) and removes duplicates.

## Method
- `public static <E> ArrayList<E> removeDuplicates(ArrayList<E> list)`: Returns a new ArrayList with unique elements from the input list.
